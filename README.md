# ANIMECAMP

Discover the Best in Anime

## Run Locally

Clone the project

```bash
  git clone https://github.com/irfanbakhtiar/animecamp.git
```

Go to the project directory

```bash
  cd animecamp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
